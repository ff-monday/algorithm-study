# 介绍

**程序 = 算法 + 数据结构**



建这个仓库的目的，一方面想真实的记录自己一步一步学习算法的过程，另一方面等到1年、2年、5年、10年后再回看或许会有温故而知新的感觉。



经过最近的练习，渐渐发现算法确实很锻炼人的**逻辑思维**能力，在写代码的过程中，也能真实发现**coding能力**的提升。



我们都深知，算法的练习是无穷无尽的，一旦开始，就没有结束，给每一个准备学习算法、正在学习算法的自己说声加油，**一起加油，奥利给**。
